# Marketplace

Django marketplace

## Stack

- Django / PostgreSQL
- Docker / Docker Compose
- Kubernetes / Helm
- Nginx Ingress
- GitHub Actions
- Terraform / Ansible
- Prometheus / Grafana
- Loki / Promtail
- RabbitMQ / Celery
- MinIO / S3

## Features

- User authentication
- Product catalog
- Search and filters
- Cart and checkout
- Orders
- Reviews
- Seller product listing

## Goal

The project demonstrates how a Django application can be containerized, deployed to Kubernetes, automated with CI/CD, monitored, logged, and prepared for cloud-native production environments
